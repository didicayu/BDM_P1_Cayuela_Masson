# P1 Final Delivery LaTeX Report

This folder contains the final P1 deliverable document.

## Layout

- `src/p1_final_delivery.tex`: main LaTeX source
- `build/`: generated PDF and auxiliary files
- `Makefile`: build automation

## Build

From the repository root:

```bash
cd docs/p1_final_delivery
make
```

Output PDF:

```bash
docs/p1_final_delivery/build/p1_final_delivery.pdf
```

## Clean

```bash
cd docs/p1_final_delivery
make clean
```
